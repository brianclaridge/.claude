# Rule 060: context7

> Use Context7 MCP for library documentation.

## Priority

Domain-specific (050-099)

## Directive

Use Context7 for ANY task involving external libraries/frameworks.

## When to Use

- Implementing features with external libraries
- Debugging library-specific issues
- Writing framework code (React, FastAPI, etc.)
- Checking API signatures
- Finding best practices/patterns

## Priority Libraries

- React/Next.js/Vue
- Python packages (pandas, FastAPI)
- Cloud SDKs (AWS, GCP, Azure)
- Database clients

## Communication

Tell user your decision:
- "Using context7 for [library] to ensure current API usage"
- "Skipping context7 - using standard library only"
