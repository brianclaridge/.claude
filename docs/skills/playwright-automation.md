# playwright-automation Skill

> Execute browser automation using Playwright Python.

## Overview

Handles complex browser automation tasks including navigation, screenshots, video recording, and data extraction.

## Activation Triggers

- "browser script"
- "record video of website"
- "extract data from webpage"
- "take screenshot of"
- browser-automation agent invocation

## Capabilities

- Page navigation
- Screenshot capture (PNG/JPEG)
- Video recording (WebM)
- Form filling
- Data extraction
- Multi-page workflows

## Output

Artifacts stored in `.data/playwright/`:
- Screenshots
- Videos
- Traces
- Extracted data

## Configuration

Playwright MCP server in `settings.json`.
