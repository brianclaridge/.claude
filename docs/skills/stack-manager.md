# stack-manager Skill

> Recommend and bootstrap application stacks.

## Overview

Analyzes project requirements and recommends appropriate technology stacks. Can bootstrap new projects.

## Activation Triggers

- "/stack-manager"
- "suggest a stack"
- "bootstrap project"
- "what stack should I use"
- "scaffold app"

## Available Stacks

| Stack | Language |
|-------|----------|
| htmx-fastapi | Python |
| nextjs-prisma | TypeScript |
| django-htmx | Python |
| go-templ-htmx | Go |
| sveltekit-supabase | TypeScript |
| vue-vite-fastapi | Python/TS |

## Workflow

1. Gather requirements
2. Analyze constraints
3. Recommend stacks
4. Bootstrap (optional)
